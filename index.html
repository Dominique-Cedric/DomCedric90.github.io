About me

Hey there! My name is Dominique, but you can call me Dom for short. I am a 2nd year Graduate employed at Rio Tinto.


I have some prior experience and exposure to a few programming languages while at university, where I got to learn and do some super cool things like:

  • * Design and build a working and responsive webpage using HTML.
  • * Build a program for a cafe menu using Python (Pycharm).
  • * Create Databases using MySQL.
  • * Build Data Models using Jupyter and Weka.
  • * Build data visualization dashboards using Power BI.


However, this wasn't enough for me to be a confident programmer at present. Given that my interest in coding still lingered in the back of my mind, I took the initiative and applied for the She Codes Plus Programming course. After a few nail biting weeks later, I was given the 'green light' and was successful in my application and was thankfully given a second chance and opportunity to get back into it and refresh my learnings.
